# Manual de Instalación – SocialFood

## 1. Requisitos previos

Para ejecutar la aplicación se necesita:

- Un servidor local:
  - XAMPP
- Un navegador web.

---

## 2. Instalación del proyecto

1. Descarga y descomprime el archivo `.zip` del proyecto.
2. Copia la carpeta completa dentro del directorio de tu servidor local:
   - En XAMPP: `htdocs/`
3. Asegúrate de no modificar la estructura interna del proyecto al moverlo.

---

## 3. Inicio de la aplicación

Para ejecutar la aplicación, abre el archivo principal del proyecto:
http://localhost/proyecto/src/www/vista/html/index.html


Este archivo cargará la página de inicio de sesión dentro del servidor local.

---

## 4. Pruebas básicas después de instalar

Una vez abierta la aplicación:

- Verificar que carga correctamente la pantalla de inicio.
- Probar el formulario de **registro de usuario**.
- Probar el formulario de **inicio de sesión**.
- Comprobar la redirección correcta según el rol:
  - Comercio → Panel de comercio  
  - Voluntario → Panel de voluntario
