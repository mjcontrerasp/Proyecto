# Manual de Instalación - SocialFood Badajoz

## Requisitos del sistema
- Servidor web: Apache/Nginx con PHP 8.x
- Base de datos: MySQL 8.x
- Certificado SSL para HTTPS
- Navegador moderno (Chrome, Firefox, Edge)

## Instalación paso a paso
1. Clonar el repositorio del proyecto.
2. Configurar la base de datos y crear las tablas según el modelo.
3. Configurar el archivo de conexión a la base de datos.
4. Colocar los archivos en las carpetas `/www`, `/src/sql` según corresponda.
5. Configurar el certificado SSL.
6. Acceder a la URL del proyecto y probar funcionalidades.

## Configuración inicial
- Crear usuarios administrativos.
- Probar el flujo de registro de comercio y voluntario.
- Verificar correo de confirmación y acceso seguro.
