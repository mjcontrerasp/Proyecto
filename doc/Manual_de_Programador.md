# Manual de Programador - SocialFood Badajoz

## Arquitectura
El proyecto se implementa con **HTML, CSS, JavaScript y PHP**.

## Estructura de carpetas
/doc # Documentación del proyecto
/src/sql # Scripts de base de datos
/www
/controlador # Archivos PHP de control
/css # Archivos de estilos CSS
/js # Scripts JavaScript
/modelo # Lógica de negocio y acceso a base de datos
/vista # Archivos HTML/PHP de interfaz
index.php # Entrada principal del sistema
config.php # Configuración del proyecto
README.md # Información general del proyecto

## Desarrollo
- Editar los archivos en `/www` según corresponda.
- Scripts SQL en `/src/sql` para inicialización de base de datos.
- Estilos CSS en `/www/css`.
- Funcionalidades JS en `/www/js`.
- Controladores PHP en `/www/controlador`.
- Modelos PHP en `/www/modelo`.
- Vistas en `/www/vista`.

## Buenas prácticas
- Separar lógica de presentación, negocio y datos.
- Usar comentarios claros en código.
- Mantener archivos organizados según la estructura.
- Validar entradas del usuario y proteger contra inyección SQL.
- Cumplimiento de RGPD y medidas de seguridad básicas.
